keccak
~~~~~~


.. function:: keccak(evm: ethereum.istanbul.vm.Evm) -> None
   :noindexentry:

   Pushes to the stack the Keccak-256 hash of a region of memory.

   This also expands the memory, in case the memory is insufficient to
   access the data's memory location.

   :param evm: The current EVM frame.

   :raises ~ethereum.istanbul.vm.error.StackUnderflowError: If `len(stack)` is less than `2`.
